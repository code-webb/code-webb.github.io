Union operation

The UNION operation is used to combine the results of two or more SELECT statements without returning any duplicate rows.

The SQL UNION operator is used to combine the result sets of 2 or more SELECT statements. It removes duplicate rows between the various SELECT statements.Each SELECT statement within the UNION must have the same number of fields in the result sets with similar data types.


  • Syntax Of UNION OPERATION

    • Mysql> SELECT Column_Name_1 FROM Table_Name_1
    UNION
    SELECT Column_Name1 FROM Table_Name_2 ;


  • Example of UNION OPERATION

    • select * from student;
    select * from city;

    select id from student union select cid from city;
