Create table with data from another table

To create a table from another we will used to create command with select command.We can create a copy of an existing table using the create table command. The new table gets the same column signature as the old table. We can select all columns or some specific columns.If we create a new table using an old table, the new table will be filled with the existing value from the old table.

  • Syntax Of Create table with data from another table

    • Mysql> CREATE TABLE newtable_name AS SELECT * FROM oldtable_name [WHERE] ;

  • Example of Create table with data from another table

    • create table s1 as select * from student;

    select * from s1;
