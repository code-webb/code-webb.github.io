Special operators in sub queries

In Sql, there are some special operators used in subqueries to perfrom specific tasks. These operators are :
(i) EXISTS
(ii) ANY
(iii) SOME
(iv) ALL


1. EXISTS : This operators is used to check for the existence of values. It produces a BOOLEAN result (TRUE/FALSE). It takes a subqauery as an argument and evalutes it to TRUE if the subquery produces any output and FALSE, if the subquery does not not produce any output.

  • Example of EXISTS operator

    • select * from person

    where person_age=34 and EXIST (select count(*) from person where personID=34);





    2. ANY : The any operator compares the lowest value from the set .

  • Example of ANY operator

    • select person_name from person

    where person_age > ANY (select person_age from person);





    3. SOME : The some operator works same like ANY operator i.e. It compares the lowst value from the set. this means both ANY and SOME operator are equivalent.

  • Example of SOME operator

    • select person_name from person

    where person_age > SOME (select person_age from person);





    4. ALL : The all operator compares the highest value from the set.

  • Example of ALL operator

    • select person_name from person

    where person_age > ALL (select person_age from person wher person_age=34);
