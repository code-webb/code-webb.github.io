Group function (aggregrate function)

Group function is also known as Aggregate functions perform a variety of actions such as counting all the rows in a table, averaging a column's data, and summing numeric data. Aggregates can also search a table to find the highest "MAX" or lowest "MIN" values in a column. As with other types of queries, you can restrict, or filter out the rows these functions act on with the WHERE clause

There are some of Group function :

1. COUNT() : COUNT function is used to Count the number of rows in a database table. It can work on both numeric and non-numeric data types.

COUNT(*) : COUNT function uses the COUNT(*) that returns the count of all the rows in a specified table. COUNT(*) considers duplicate and Null.

  • Syntax of count() Function

    • Mysql> SELECT count(columName) FROM tableName WHERE condition;


  • Example of count() Function

    • select * from employee;
    select count(emp_id) from employee where emp_id < 104;





    2. SUM() : Sum function is used to calculate the sum of all selected columns. It works on numeric fields only.

  • Syntax of sum() Function

    • Mysql> SELECT sum(columName) FROM tableName WHERE condition;


  • >Example of sum() Function

    • select * from employee;
    select sum(emp_salary) from employee;





    3. AVG() : The AVG function is used to calculate the average value of the numeric type. AVG function returns the average of all non-Null values.

  • Syntax of avg() Function

    • Mysql> SELECT avg(columName) FROM tableName WHERE condition;


  • Example of avg() Function

    • select * from employee;
    select avg(emp_salary) from employee where emp_name in ("mohan","Nisha","Sanjeet");





    4. MAX() : MAX function is used to find the maximum value of a certain column. This function determines the largest value of all selected values of a column.

  • Syntax of max() Function

    • Mysql> SELECT max(columName) FROM tableName ;


  • Example of max() Function

    • select * from employee;
    select max(emp_salary) from employee;





    5. MIN() : MIN function is used to find the minimum value of a certain column. This function determines the smallest value of all selected values of a column.

  • Syntax of min() Function

    • Mysql> SELECT min(columName) FROM tableName;


  • Example of min() Function

    • select * from employee;
    select min(emp_salary) from employee;
