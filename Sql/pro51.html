Introduction to Nested Query

Nested Query is a query with in another query. SQl has an ability to nest queries with in one another . A subquery is a SELECT Statement that is nested with in another SELECT statement and which returns intermediate results. The result of Subquery or inner query is dynamically substituted in the condition of outer query. SQL first evalutes the inner query or subquery with in the WHERE clause. The result of inner query is then substitued in the condition of the outer query.
Most of the time ,a subquery is used when you know how to search for a value using a SELECT statement, but do not know the exact value.

    select person_name from person where person_age =(select person_age from person where person name="aman");




    select person_name from person where person_age =(select max(person_age ) from person);

    select person_name, person_age where person_age in (select min(person_age ) from person group by person_age);
