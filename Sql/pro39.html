Rollback Command

ROLLBACK : To user can continue with any number of insert, updates and /or deletion, and still can undo the work by issuing the ROLLBACK statement. The ROLLBACK statement is used to discard parts or all the work the user has done in the current transaction.
ROLLBACK is important in case an error is detected during the DML (data manipulation language) operations.

  • Syntax Of Rollback command

    • Mysql> ROLLBACK;


  • Example of Rollback command

    • select * from city;

    delete from city where cid=3;

    select * from city;






    ROLLBACK;

    select * from city;
