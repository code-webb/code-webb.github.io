Update statement

The SQL commands (UPDATE and DELETE) are used to modify the data that is already in the database. The SQL DELETE command uses a WHERE clause.

UPDATE statement is used to change the data of the records held by tables. Which rows is to be update, it is decided by a condition. To specify condition, we use WHERE clause.

  • Syntax Of Update statement

    • Mysql> UPDATE table_name SET [column_name1= value1,... column_nameN = valueN] WHERE condition ;

  • Example of Update statement

    • update student set rollno=1 where rollno=2;
