Create table

Create Table : Creating a basic table involves naming the table and defining its columns and each column's data type.CREATE TABLE is the keyword telling the database system what you want to do. In this case, you want to create a new table. The unique name or identifier for the table follows the CREATE TABLE statement.Then in brackets comes the list defining each column in the table and what sort of data type it is.

The SQL CREATE TABLE statement is used to create a new table.

  • Syntax Of Create Table

    • Mysql> CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype,
    );

  • Example of Create table

    • create table student(
    Rollno int,
    Name varchar(20),
    Age int,
    Marks int
    );
